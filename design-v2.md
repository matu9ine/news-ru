КОНЦЕПЦИЯ v2: "ХОЛОДНАЯ ПРЕМИАЛЬНАЯ РЕДАКЦИЯ" — РАСШИРЕННАЯ

База та же:
Financial Times (структура) + РБК (строгость) + Apple News (чистота)
+ The New York Times (редакционная иерархия)
+ Bloomberg (информационная плотность)

Ключ (усилен):
→ контент важнее интерфейса
→ интерфейс невидим, пока не нужен
→ каждый пиксель работает на читаемость

───────────────────────────────────

1. ФИЛОСОФИЯ

Три правила, которые решают всё:

1. Воздух дороже декора
   → пустое пространство = премиум
2. Типографика — это UI
   → 80% ощущения "дорого" даёт шрифт, не цвет
3. Движение только осмысленное
   → никаких анимаций "для красоты"

Если элемент не помогает читать — его нет.

6. HERO (редакционный)

Сетка осталась 65/35, но:

Левая часть:
  изображение 16:9 (а не 3:2 — кинематографичнее)
  тонкая подпись под фото: автор · источник
  overlay-рубрика в левом верхнем углу фото

Правая часть:
  метка типа материала (НОВОСТЬ / АНАЛИЗ / МНЕНИЕ)
  заголовок fs-4xl, weight 700
  лид fs-md, text-secondary
  строка метаданных: рубрика · дата · время чтения
  иконка "обновлено" с тултипом

Варианты hero:
  A. Одна главная + 3 вторичные  (стандарт)
  B. Две равные новости          (раскол повестки)
  C. Full-bleed с текстом поверх (особые события)

Правило: hero не листается, не анимируется, не крутится.
Карусели убивают премиальность.

9. СТРАНИЦА СТАТЬИ (сердце проекта)

Шапка статьи:
  рубрика (КАПС, accent)
  заголовок (fs-4xl, центрирован или left)
  подзаголовок (fs-md, text-secondary, max 640px)
  автор (аватар 40px + имя + должность)
  дата публикации + дата обновления
  время чтения ("~8 мин")
  счётчик комментариев
  кнопки шейринга (иконки, без подписей)

Главное изображение:
  full-bleed или в container-wide
  подпись снизу: italic, text-secondary, fs-sm
  источник фото в конце подписи

Тело статьи:
  max-width: 680px
  font-size: 18px (не 16 — крупнее читается премиальнее)
  line-height: 1.75
  параграфы: margin-bottom 24px
  первый параграф: fs-md, weight 500 (лид)

Типографические детали:
  → drop cap: первая буква 4 строки, Playfair, accent
  → блок-цитата: отступ слева 32px, линия 3px accent,
                 italic, fs-lg
  → списки: маркеры accent, отступ 24px
  → сноски: fs-sm, text-tertiary, номера надстрочные
  → ссылки в тексте: подчёркивание с offset 3px
  → изображения внутри: full-width в пределах 680px
  → "врезка" (aside): серый фон, padding 32px,
                       текст fs-base, заголовок fs-lg

Спецэлементы:
  ▸ Хронология события (timeline)
  ▸ Ключевые цифры (крупные Playfair числа + подпись)
  ▸ "Что важно знать" — буллет-врезка сверху
  ▸ "Контекст" — серая плашка с фоном событий
  ▸ Связанные материалы (3 карточки в конце)

Навигация чтения:
  → прогресс-бар чтения сверху (2px, accent)
  → sticky-блок "Поделиться" слева (desktop)
  → кнопка "Наверх" появляется после 50% скролла
  → якоря для подзаголовков (#копировать-ссылку)

10. СОСТОЯНИЯ И ОБРАТНАЯ СВЯЗЬ

Loading:
  skeleton вместо спиннера
  формы блоков повторяют контент
  shimmer очень медленный (2s)

Empty:
  иконка + одна строка + одна кнопка
  без иллюстраций-персонажей

Error:
  маленькая плашка сверху
  красная полоса слева 3px
  текст + кнопка "Повторить"

Success:
  тост снизу-справа
  зелёная линия слева
  исчезает через 4s

Focus (важно):
  outline: 2px solid accent
  outline-offset: 2px
  НИКОГДА outline: none без замены

11. ДВИЖЕНИЕ

Тайминги:
  --dur-fast:   120ms   hover, фокус
  --dur-base:   200ms   большинство
  --dur-slow:   300ms   layout, дропдауны
  --dur-slower: 500ms   страничные переходы

Кривые:
  --ease-out:     cubic-bezier(0.16, 1, 0.3, 1)   выход
  --ease-in-out:  cubic-bezier(0.65, 0, 0.35, 1)  симметрия
  НЕ использовать: ease, linear (скучно)

Правила:
  → только transform и opacity (GPU)
  → никаких bounce, elastic, spring
  → prefers-reduced-motion → все анимации 0ms
  → параллакс запрещён

17. ДОСТУПНОСТЬ (WCAG AA)

  контраст текста ≥ 4.5:1
  контраст крупного текста ≥ 3:1

18. ДЕТАЛИ УРОВНЯ "ОНИ ЗАМЕТЯТ"

Что отличает сайт за 50к от сайта за 500к:

  ✓ одинаковый вертикальный ритм везде
  ✓ оптическое выравнивание заголовков
  ✓ кавычки ёлочкой «…» для основного
  ✓ апостроф ’ вместо '
  ✓ неразрывные пробелы в нужных местах
  ✓ числа с tabular-nums в списках
  ✓ курсив только для названий и терминов
  ✓ одинаковые радиусы скругления (4 / 8 / 12)
  ✓ одинаковые тени (или отсутствие теней)
  ✓ иконки одного набора (Lucide / Heroicons)
  ✓ одинаковая толщина линий (1px / 2px)
  ✓ hover-состояния у ВСЕХ интерактивных
  ✓ disabled-состояния у ВСЕХ кнопок
  ✓ одинаковая длительность анимаций
  ✓ фокус-стили не "выключены"
